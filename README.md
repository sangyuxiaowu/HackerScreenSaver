# Hacker Screen Saver

.NET develops a screen saver. 

相关文章：[《用.NET设计一个假装黑客的屏幕保护程序》](https://mp.weixin.qq.com/s/WAsWQINJA3cletktYb-TKw)

已添加设置功能，可以选择本地网页也可以设置自定义的 URL 地址。

软件自身提供 2 个网页屏保。


### 默认 Hacker Typer

![Hacker Typer](doc/2.png)

来自古老版本的 [NEO Hacker Typer](http://hackcode.ishoulu.com/scp/) 并做了 IE 适配。

更多效果可前往 [GeekTyper](https://geektyper.com/)
